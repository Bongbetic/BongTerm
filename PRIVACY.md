# BongTerm Privacy Notice

BongTerm is local-first. Telemetry is off by default.

## Defaults

- No cloud account is required.
- Diagnostic export is local unless the user explicitly shares it.
- Export flows must show a redaction preview before sharing.
- Secret redaction applies to persisted/exported text, not raw terminal display.

## Diagnostic Export May Include

- BongTerm logs and crash metadata.
- Command history snippets after redaction preview.
- Process/resource metrics.
- OS/hardware metadata needed for debugging.

## Not Included By Default

- Raw secrets.
- Cloud upload.
- Silent telemetry.
