## Context

Working name "BongTerm" may have negative connotations in some markets ("bong" = smoking paraphernalia). This ADR captures the trademark search and brand-perception review outcome required before the public flip.

## Required before public release (Phase 6.C)

1. USPTO trademark search for "BongTerm" and "BongT".
2. EUIPO trademark search.
3. Indian TM database search.
4. GitHub / npm / crates.io / domain availability check.
5. Brand perception review document (`docs/adr/0002-brand-perception-notes.md`).
