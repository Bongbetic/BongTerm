# Narrator Smoke: Gate 18

1. Install or run BongTerm on Windows 11.
2. Start Narrator.
3. Focus the main window.
4. Confirm Narrator can enumerate window, tab strip, active pane, terminal text, scrollback, command blocks, and main controls.
5. Type through an IME and confirm preedit text appears at the cursor cell.
6. Move the window between monitors with different DPI and confirm text remains aligned.
